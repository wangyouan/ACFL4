# Capital Structure and Dividend Policy Lecture Slides

Lecture slides for the "Capital Structure and Dividend Policy" course offered at WISE and SOE, Xiamen University (Spring 2026).

## Overview
This repository contains LaTeX source files for lecture slides covering key topics in corporate finance:

- Efficient capital markets and behavioral challenges
- Capital structure theories (Modigliani-Miller, trade-off theory)
- Valuation methods for levered firms (APV, FTE, WACC)
- Dividend policy and payout strategies
- Chinese market applications

## Key Topics Covered
1. **Market Efficiency**
   - EMH foundations
   - Behavioral challenges
   - Chinese market evidence

2. **Capital Structure**
   - MM propositions
   - Trade-off theory
   - Agency costs
   - Pecking order theory

3. **Valuation Methods**
   - Adjusted Present Value (APV)
   - Flow-to-Equity (FTE)
   - Weighted Average Cost of Capital (WACC)

4. **Dividend Policy**
   - Payout methods
   - Dividend irrelevance theory
   - Real-world considerations
