# Capital Structure and Dividend Policy Lecture Slides


Lecture slides for "Capital Structure and Dividend Policy" course offered at WISE & SOE, Xiamen University (Spring 2025)

This repository contains LaTeX source files and compiled lecture slides covering key topics in corporate finance:

- Efficient Capital Markets & Behavioral Challenges
